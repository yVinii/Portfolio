Site de Imobiliária

Site da Imobiliária LSilva foi um projeto no qual foi feito uma modernização completa no site, com funções únicas como a parte de visualização dos clientes, até um sistema de login e painel de controle completo.

Site de Imobiliária

O projeto foi feito com Html e Css original e responsivo para o frontend, Javascript e Node.js para o backend, com implementação de banco de dados MySQL, além de uma versão mobile em React-Native, foi completamente documentado, e feito com metodologias ágeis de Scrum e Kanban.

Criação de Campeonatos

GameON é um trabalho que visa facilitar a modelagem de campeonatos, trazendo verificação por cpf para os jogadores, sorteio, chaveamento para campeonatos e relatório em um PDF ao fim do campeonato.

Criação de Campeonatos

O projeto foi feito em Java, suas funcionalidades como verificação de cpf, registro de jogadores, times, campeonatos, sorteio e geração de pdf para resultados são vinculadas com banco de dados MySQL, com a metodologia ágil Scrum, além de toda sua documentação completa.
